Queue & Until Programs written by Andy Sayler

EXTRA CREDIT: resolver threads are based off of number of CPU cores using sysconf(_SC_NPROCESSORS_ONLN)

To build file use the command "make"
ex: make

To run file use the command "./multi-lookup" followed by input text files with spaces in between
ex: ./multi-lookup input/names1.txt input/names2.txt

To clean up output files use the command "make clean"
ex: make clean
